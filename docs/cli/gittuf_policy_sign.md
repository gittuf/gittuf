## gittuf policy sign

Sign policy file

### Synopsis

This command allows users to add their signature to the specified policy file.

```
gittuf policy sign [flags]
```

### Options

```
  -h, --help                 help for sign
      --policy-name string   name of policy file to sign (default "targets")
```
