## gittuf setup

Launch the gittuf setup wizard to quickly get started with gittuf on your repository

### Synopsis

The 'setup' command serves as an alternative to the manual setup process for gittuf, intended for rapid deployment of gittuf on repositories with a basic security policy

```
gittuf setup [flags]
```

### Options

```
      --github-token string   GitHub token for enabling gittuf GitHub approvals
  -h, --help                  help for setup
```
